# Laloli Studio
Laloli studio es un estudio de diseño de interiorismo especializado en cuartos de bebés y niños. Tienen un estilo muy neutral, clásico, fresco y elegante. Tiene una paleta de colores muy sencilla pero clásica que resalta las imágenes de cada proyecto.

# Objetivo del proyecto
Crear un landing page para laloli studio.
El landing page debe guiar a los clientes a contactar a laloli por WhatsApp
También debe mostrar el portafolio de laloli
Debe haber un link a instagram: @lalolistudio
Utiliza las imágenes de Assets
Puedes basarte en Mockup.jpg
